# Requirements

- [http-server](https://www.npmjs.com/package/http-server)

# Instructions


## For V1,V2 examples (websockets)

Start websocket server

```bash
cd ws_server
node index.js
```

## For V3, V5

Start Static Web server
```bash
http-server .
```

By default it starts on [http://localhost:8080](http://localhost:8080)
